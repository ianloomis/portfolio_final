PROJECT BRIEF

Every year, Apple comes to R.I.T to host its annual iOS challenge. In a fast-paced four day timeline, teams of multi-disciplinary students develop applications for Apple devices for judging by Apple executives and employees. Each year presents a new theme for which the projects must be based. For this particular challenge, the theme was multi-platform development. There is no specific genre the application has to fall under, but it must include use by at least two different mediums (mobile and smart TV, mobile and web, etc.)

For this project, I oversaw the design of the front end of both the mobile and web platforms. I worked with a software engineer and a full-stack web developer to create an interactive model of our application, MeetMe. MeetMe is a meeting app that uses Google Map's API to allow users to pin meeting places on campuses. Features include allowing users to set a timeframe and location for the meeting, as well as organize meeting places into specific categories such as Math or Video Games.

The goal of this application was to combine web and mobile features to allow users to easily set up study groups, tutoring sessions, or meet others with common interests. Combining both the ability to browse and post on meeting sites through mobile mediums, creating a web dashboard allows users to have the ability to browse or post from any computer without proprietary Apple hardware or software.
